## Description

Reference table of 38 travel destinations across three regions: us_atlantic (14 locations from FL to NY), gulf_coast (6 locations in TX/AL/FL/LA), and caribbean (18 international locations). Includes lat/lon coordinates for geographic analysis, airport_code, and avg_hotel_price_per_night. Gulf Coast destinations are key for hurricane catastrophe exposure modeling.
